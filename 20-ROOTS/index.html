N.20 Домашнее задание ROOTS

Найти ошибку в функции нахождения корней квадратного уравнения

В функции squareRoots была доработана обработка вырожденных случаев. В первом случае, когда a = 0 и b ≠ 0, уравнение становится линейным (b·x + c = 0), и функция возвращает единственный корень -c / b в виде массива.

Во втором случае, когда a = 0, b = 0 и c = 0, уравнение имеет бесконечно много решений, поэтому корректное значение вернуть невозможно; в этом случае функция выбрасывает исключение, которое обрабатывается с помощью try/catch.
